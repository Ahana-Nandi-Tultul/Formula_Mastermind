What is DOM ? What is the purpose of DOM?

Ans: DOM means Document Object Model which is the logical structure of the HTML documents that makes it easy to manipulate the HTML documents. The purpose of DOM is to create, remove or change elements of the document.

How will you select HTML elements using DOM? Name The DOM Methods

Ans: There are five ways to select a HTML element using DOM. Those are:

  1. getElementsByTagName()
  2. getElementById()
  3. getElementsByClassName()
  4. querySelectorAll()
  5. querySelector
